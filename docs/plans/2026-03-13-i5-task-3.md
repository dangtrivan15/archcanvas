## Description

This task creates the CLI's shared infrastructure — the plumbing that every command uses.

### CLIError (Contract C4.6, C4.7)

Simple error class with a `code` string field. Error codes include `PROJECT_NOT_FOUND`, `PROJECT_EXISTS`, `PROJECT_LOAD_FAILED`, `UNKNOWN_NODE_TYPE`, `INVALID_ARGS`, plus engine error codes forwarded as-is.

### Output Formatting (Contract C6)

Two functions:
- `formatSuccess(data, { json })` — when `json: true`, returns `JSON.stringify({ ok: true, ...data })`. When `json: false`, returns human-readable text.
- `formatError(error, { json })` — when `json: true`, returns `{ ok: false, error: { code, message } }`. When `json: false`, returns `Error: [CODE] message`.

Error output goes to stderr, success to stdout. Exit code 0 on success, 1 on error.

### Context Loading (Contract C4)

`loadContext(projectPath?)`:
1. If `projectPath` provided, use it. Otherwise walk cwd upward looking for `.archcanvas/` directory.
2. `createFileSystem(resolvedPath)` → `NodeFileSystem`
3. `fileStore.openProject(fs)` and wait for `status === 'loaded'`
4. `registryStore.initialize()` (builtins loaded from TS objects)
5. Return `{ fs }`. Commands access data via `fileStore.getState()`.

`resolveCanvasId(scope?)`:
- `undefined` or `'root'` → `ROOT_CANVAS_KEY` (`'__root__'`)
- Any other value → pass through as-is

### Commander Entry (Contract C11)

`src/cli/index.ts` sets up the commander program with:
- Program name: `archcanvas`
- Version from package.json
- Global `--json` flag
- 9 subcommands registered (initially as stubs that throw "not implemented" — filled in by Tasks 4–6)
- Top-level error handler that catches CLIError and unhandled errors, formats them, and exits with code 1

### Testing Strategy

**context.test.ts:** Use InMemoryFileSystem (or temp dirs) with `.archcanvas/main.yaml`. Test: explicit path loading, cwd upward search, PROJECT_NOT_FOUND, PROJECT_LOAD_FAILED, resolveCanvasId mapping. Reset stores between tests.

**output.test.ts:** Pure function tests. Verify JSON output is parseable, shapes match contracts, human output is non-empty strings.
