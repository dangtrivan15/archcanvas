## Description

Final P05 cleanup task. After P05-T1 created the unified store with a compatibility
facade, and P05-T2 updated the core canvas hooks, this task migrates all remaining
consumers and deletes the facade.

### Consumer Migration

For each file, the change is mechanical — replace the import and update the selector:

**Pattern A — Direct import replacement:**
```typescript
// Before
import { useNestedCanvasStore } from '@/store/nestedCanvasStore';
const depth = useNestedCanvasStore(s => s.fileStack.length);

// After
import { useNavigationStore } from '@/store/navigationStore';
const depth = useNavigationStore(s => s.fileDepth);
```

**Pattern B — getState() calls:**
```typescript
// Before
const nestedStore = useNestedCanvasStore.getState();
nestedStore.popToRoot();

// After
const navStore = useNavigationStore.getState();
navStore.goToRoot();
```
