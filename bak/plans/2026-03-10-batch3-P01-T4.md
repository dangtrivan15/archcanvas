### useAppInitialization Hook

Extract these 6 useEffects from App.tsx into `src/hooks/useAppInitialization.ts`:

1. **Engine initialization** (App.tsx ~line 185) ‚Äî `initialize()` on mount
2. **Splash screen hide** (App.tsx ~line 191) ‚Äî After `initialized` becomes true
3. **Registry bridge setup** (App.tsx ~line 201) ‚Äî After initialized, with registry
4. **URL file loading** (App.tsx ~line 208) ‚Äî After initialized, loads from URL param
5. **Panel width responsiveness** (App.tsx ~line 164) ‚Äî Viewport width changes
6. **Auto-open right panel on selection** (App.tsx ~line 179) ‚Äî Selection changes

The hook returns `{ initialized }` so App.tsx can conditionally render.

**Ordering dependencies between effects:**
- Effects 2, 3, 4 depend on `initialized` (from effect 1)
- Effect 3 depends on `registry` (available after initialization)
- Effects 5, 6 are independent
